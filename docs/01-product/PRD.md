# PRD: 중학생 학업 성취 대시보드

- Version: v1.0
- Date: 2026-02-20
- Product Stage: Pre-development

## 1. 배경과 목적

대한민국 중학생이 현재 수학 학업 성취 상태를 스스로 점검하고, 약점을 인식해 보완하는 메타인지 중심 대시보드 웹서비스를 제공한다.

## 2. 목표

- 현재 날짜 기준 학년/학기 진도와 개인 성취도를 한눈에 제공
- 오답을 축적하여 단원별 취약점을 시각적으로 분석
- 중학생 친화 UI/UX 제공
- 1단계 수학 MVP 후 과목 확장 가능 구조 확보
- 중학교 시작, 초등/고등 확장 가능한 데이터 구조 적용

## 3. 사용자

- 1차 사용자: 보호자 + 중학생 1명(딸)
- 2차 사용자: 다수 학생 + 보호자

## 4. 범위

### In Scope (MVP)

- 수학 과목만 제공
- 학년/학기/단원 진도표 조회
- 문제집/학습지별 정오답 입력
- 오답 문제 이미지 업로드
- 오답 카테고리 분류
  - 단순 연산 실수
  - 문제 잘못 읽음
  - 문제 이해 못함
- 대시보드 시각화
  - 진도 대비 성취도
  - 단원별 약점
  - 주간/월간 변화

### Out of Scope (MVP 제외)

- 실시간 학교 LMS 연동
- 자동 채점 OCR 고도화
- 다과목 동시 정교 분석

## 5. 핵심 사용자 시나리오

1. 학생/보호자가 학년과 학기를 선택한다.
2. 오늘 날짜 기준 권장 진도 대비 현재 진도를 확인한다.
3. 문제집(예: 디딤돌 중학 수학 1학기) 단원별 정오답을 입력한다.
4. 틀린 문제 사진을 업로드하고 오답 카테고리를 지정한다.
5. 대시보드에서 약점 단원, 오류 유형 비중, 재학습 우선순위를 확인한다.

## 6. 기능 요구사항

- FR-001: 사용자 계정 및 학생 프로필 관리
- FR-002: 교육과정(학교급/학년/학기/과목/단원) 조회
- FR-003: 문제집/학습지 등록 및 단원 맵핑
- FR-004: 시도(Attempt)별 정오답 입력
- FR-005: 오답 이미지 업로드 및 조회
- FR-006: 오답 카테고리(복수 선택 가능) 저장
- FR-007: 대시보드 시각화(진도, 성취도, 약점)
- FR-008: 기간별 리포트(주간/월간)
- FR-009: 약점 우선순위 추천(정답률, 최근성, 반복오류 반영)

## 7. 비기능 요구사항

- 모바일 우선 반응형 (iPhone/iPad/PC)
- 맥 미니 로컬 호스팅 우선
- 개인정보 최소 수집
- 이미지 업로드 안정성 및 백업
- 첫 화면 로딩 3초 이내 목표

## 8. KPI

- 주 3회 이상 사용률
- 오답 기록 누적 수
- 약점 단원 재학습 후 정답률 향상
- 4주 유지율

## 9. 확장성 요구사항

모든 커리큘럼/학습 데이터에 아래 축을 유지한다.

- school_level: elementary | middle | high
- subject: math (MVP), future multi-subject
- grade, semester, unit hierarchy

## 10. 배포/운영 원칙

- Phase 1: Mac mini local self-hosting
- Phase 2: 필요 시 외부 호스팅(Vercel/Render 등) 이관
- 컨테이너 기반으로 이관 비용 최소화
