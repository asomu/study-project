# Session Handoff Guide

## 세션 시작 체크리스트

1. `/Users/mark/Documents/project/study-project/docs/05-operations/PROJECT_STATUS.md` 확인
2. `/Users/mark/Documents/project/study-project/docs/05-operations/DECISION_LOG.md`의 최신 결정 확인
3. 현재 작업 범위가 PRD와 일치하는지 점검

## 세션 종료 체크리스트

1. 완료 항목을 `PROJECT_STATUS.md`에 반영
2. 변경된 결정이 있으면 `DECISION_LOG.md` 추가
3. 다음 세션의 첫 작업 1~3개를 `Next Actions`에 기록
