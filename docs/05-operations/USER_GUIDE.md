# User Guide

- Last Updated: 2026-04-12
- Scope: Wrong-note-first + workbook-progress private beta
- Audience: 보호자 / 학생 / 운영 지원자
- Goal: 현재 제품의 실제 사용 순서를 역할별로 빠르게 이해하고 따라갈 수 있게 한다.

## 1. 역할별 진입점

- 보호자
  - 첫 가입: `/signup`
  - 로그인: `/login`
  - 첫 설정: `/students/manage`
  - 일상 사용 홈: `/dashboard`
- 학생
  - 첫 활성화: `/student/activate`
  - 로그인 후 홈: `/student/dashboard`

메모:

- 학생/보호자 보호영역은 공통 사이드바 셸을 사용한다.
- desktop은 좌측 사이드바, 모바일은 상단 `메뉴 열기` 버튼으로 같은 탐색 구조를 연다.
- 대시보드 안의 `개요 / 문제집 진도 / 오답 그래프 / 오답 탐색` 같은 섹션도 사이드바에서 바로 이동할 수 있다.

## 2. 보호자 첫 설정 순서

1. `/signup` 또는 `/login`으로 보호자 계정에 들어간다.
2. `/students/manage`에서 학생 프로필을 만든다.
3. 같은 화면에서 `초대코드 발급`을 눌러 학생 초대코드를 받는다.
4. 학생에게 `/student/activate` 주소와 초대코드를 전달한다.
5. 학생이 첫 계정을 활성화하면 이후부터는 `/dashboard`에서 학생을 선택해 관리한다.

메모:

- seed 보호자 계정은 기본 제공되지만 학생 로그인 계정은 자동으로 만들어지지 않는다.
- 학생 로그인에 문제가 생기면 `계정 재설정 후 새 초대코드 발급`으로 다시 연결할 수 있다.

## 3. 보호자 일상 사용 흐름

### 3.1 학생 선택

1. `/dashboard` 진입
2. 상단 sticky bar의 `학생 선택`에서 관리할 학생을 고른다.
3. 필요하면 같은 bar에서 `학생 보기` 또는 `문제집 관리` 모드를 전환한다.

### 3.2 문제집 템플릿 등록

1. `문제집 진도` 섹션의 `문제집 템플릿 등록` 카드 확인
2. 아래 항목 입력
  - 문제집 이름
  - 출판사
  - 학교급
  - 대상 학년
  - 학기
  - 단계 목록
3. `템플릿 저장` 클릭

현재 동작:

- 단계는 1개 이상 필요하다.
- 단계 순서는 직접 조정할 수 있다.
- 템플릿 배정 후에도 제목/출판사/활성 상태는 수정 가능하다.
- 제목/출판사 수정 UX는 현재 inline editor 기반이다.

### 3.3 학생 문제집 배정

1. 같은 섹션의 `학생 문제집 배정` 카드에서 템플릿 선택
2. `학생에게 배정` 클릭
3. 필요하면 `배정 보관` 또는 `배정 복구`로 활성 상태를 바꾼다.

### 3.4 학생 진도 보기

1. 대상 학년과 문제집 선택
2. KPI 확인
  - 전체 단계 수
  - 완료
  - 진행중
  - 시작전
  - 완료율
3. 단원별 완료 단계 수 bar chart 확인
4. `단원 x 단계` matrix에서 현재 상태를 확인

메모:

- 보호자도 matrix 셀을 눌러 상태를 직접 바꿀 수 있다.
- 학생과 보호자는 같은 진도 데이터를 본다.

### 3.5 학생 오답 보기와 피드백 남기기

1. `오답 현황 그래프`에서 그래프 기준, 대상 학년, 학기 변경
2. 아래 KPI와 카드 목록으로 원하는 wrong-note를 찾는다.
3. 상세를 열고 `보호자 피드백` 작성
4. `피드백 저장` 클릭

메모:

- 빈 값으로 저장하면 기존 피드백을 지울 수 있다.
- 피드백 완료 건수는 KPI에 반영된다.
- 왼쪽 사이드바에서 `학생 컨텍스트`, `문제집 진도`, `오답 그래프`, `오답 탐색` 섹션으로 바로 이동할 수 있다.

## 4. 학생 일상 사용 흐름

### 4.1 첫 계정 활성화

1. `/student/activate` 진입
2. 보호자가 준 초대코드 입력
3. 학생 로그인 아이디, 비밀번호, 표시 이름 입력
4. 활성화 완료 후 `/student/dashboard`로 이동

### 4.2 문제집 진도 업데이트

1. `문제집 진도` 섹션에서 대상 학년과 문제집 선택
2. matrix 셀을 눌러 상태 순환
  - `시작전 -> 진행중 -> 완료 -> 시작전`
3. KPI와 bar chart 변화를 확인

메모:

- 화면 왼쪽 사이드바에서 `개요`, `빠른 업로드`, `문제집 진도`, `오답 그래프`, `오답 탐색` 섹션으로 바로 이동할 수 있다.
- 모바일에서는 상단 `메뉴 열기` 버튼으로 같은 섹션 목록을 연다.

### 4.3 wrong-note 업로드

1. 이미지 파일 선택
2. 대상 학년과 학기 선택
3. 필요하면 `문제집 선택`
4. workbook을 골랐다면 `문제집 단계` 선택
5. 단원 선택
6. 오류유형 선택
7. 필요하면 학생 메모 입력
8. `오답노트 저장` 클릭

메모:

- workbook을 선택하지 않으면 일반 wrong-note로 저장할 수 있다.
- workbook을 선택하면 stage 선택이 필요하다.
- workbook을 선택한 경우 grade/semester는 해당 workbook 기준으로 고정된다.

### 4.4 wrong-note 다시 보기

1. 카드 목록에서 wrong-note 선택
2. 상세에서 아래 항목 수정 가능
  - 대상 학년
  - 학기
  - 문제집 / 단계
  - 단원
  - 오류유형
  - 사진 교체
  - 학생 메모
3. 필요 없으면 삭제 가능
4. 상세 하단에서 보호자 피드백 확인

## 5. 현재 제품 규칙

- 수학 과목만 지원한다.
- wrong-note는 `학년/학기/단원/오류유형` 기준으로 분류한다.
- 오류유형은 단일 선택이다.
- 지원 이미지 형식:
  - `png`
  - `jpg/jpeg`
  - `webp`
  - `heic/heif`
- 이미지 저장은 repo 밖 앱 데이터 루트에 보관되고 화면에는 guarded image API로만 노출된다.
- 레거시 `records/new`, `study/*`, `wrong-answers/manage`, `student/wrong-answers`는 현재 기본 홈으로 redirect된다.

## 6. 자주 보는 문제와 빠른 확인법

### 학생이 로그인할 수 없을 때

- `/students/manage`에서 초대코드가 발급됐는지 확인
- 이미 연결된 학생이면 `계정 재설정 후 새 초대코드 발급`으로 다시 연결

### 학생 홈에 문제집이 안 보일 때

- 보호자가 해당 학생에게 workbook을 실제로 배정했는지 확인
- 배정이 `보관` 상태인지 확인

### wrong-note 저장 시 단계 선택이 막힐 때

- 먼저 `문제집 선택`을 해야 한다.
- workbook을 선택하지 않은 경우에는 단계 선택이 비활성 상태가 정상이다.

### 이미지가 안 보일 때

- 파일이 누락되었을 수 있다.
- 다시 업로드하면 현재 wrong-note에 새 이미지로 연결할 수 있다.

### 학생/보호자 화면이 서로 다른 이유

- 학생은 자기 wrong-note를 생성/수정/삭제한다.
- 보호자는 조회와 피드백 중심이다.
- workbook progress는 두 역할 모두 수정 가능하다.
