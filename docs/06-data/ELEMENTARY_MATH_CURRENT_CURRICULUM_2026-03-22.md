- 공식 성취기준은 `1~2학년`, `3~4학년`, `5~6학년` 학년군 기준으로 제시되므로, 앱의 `학년 / 학기 / 단원` 콤보박스를 위해 학기 단원명은 대표 단원 라벨로 정규화한다.
