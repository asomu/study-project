# Learning Notes (study-tech-explainer)

이 영역은 `study-tech-explainer` 스킬로 생성한 학습 설명 문서를 보관한다.

## 목적

- 구현된 기술의 선택 이유와 동작 흐름을 나중에 다시 학습할 수 있도록 기록
- 세션이 바뀌어도 동일한 설명 품질/구조를 유지

## 저장 규칙

- 설명 문서: `notes/YYYY-MM-DD-<topic>.md`
- 인덱스: `TECH_EXPLAINER_INDEX.md`에 문서 링크와 요약을 추가
- 템플릿: `templates/TECH_EXPLAINER_NOTE_TEMPLATE.md`

## 운영 규칙

- 파일 경로와 실제 코드 레퍼런스를 반드시 포함한다.
- 설명은 학습자 관점으로 작성하되, 추론과 사실을 구분한다.
- 동일 토픽이 있으면 새 문서를 무한 생성하지 않고 기존 문서를 업데이트할 수 있다.
